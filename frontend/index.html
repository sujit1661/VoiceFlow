How it works

Three steps to pure flow state

Hold a key, speak naturally, release. Flow handles everything else — transcription, polishing, and typing.

01
Hold & Speak
Press the hotkey in any app — Gmail, Slack, VS Code, anything. Start speaking naturally. No window switching, no mode changes.
02
AI Polishes Instantly
Flow strips filler words, fixes grammar, and formats your text for the context — email, code comment, Slack message — all in under a second.
03
It Just Appears
Release the key. Polished text is typed exactly where your cursor was. No copy-paste, no interruption. Pure unbroken flow.
